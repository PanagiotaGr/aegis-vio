# Experiment 004 – Ανάλυση Ποιότητας Χαρακτηριστικών (Feature Quality Analysis)

## Στόχος

Στόχος του πειράματος είναι η αξιολόγηση της ποιότητας των οπτικών χαρακτηριστικών που εξάγονται από διαδοχικά frames του EuRoC MAV Dataset.

Η ποιότητα των χαρακτηριστικών αποτελεί σημαντικό παράγοντα για την απόδοση ενός Visual-Inertial Odometry συστήματος, καθώς επηρεάζει άμεσα την ακρίβεια της εκτίμησης θέσης.

## Dataset

EuRoC MAV Dataset

Sequence:

* MH_01_easy

## Μεθοδολογία

Για 300 διαδοχικά frame pairs:

1. Εξαγωγή ORB χαρακτηριστικών.
2. Αντιστοίχιση χαρακτηριστικών μεταξύ διαδοχικών frames.
3. Υπολογισμός αριθμού matches.
4. Υπολογισμός inlier ratio μέσω γεωμετρικού ελέγχου.
5. Ορισμός δείκτη ποιότητας:

Feature Quality = Matches × Inlier Ratio

## Αποτελέσματα

* Μέσος αριθμός matches: 871
* Μέσο inlier ratio: 0.716
* Μέση ποιότητα χαρακτηριστικών: 630.73

## Συμπέρασμα

Η ακολουθία MH_01_easy παρουσιάζει σταθερή ποιότητα οπτικών χαρακτηριστικών και αποτελεί κατάλληλο benchmark για τα επόμενα στάδια του συστήματος AegisVIO.
