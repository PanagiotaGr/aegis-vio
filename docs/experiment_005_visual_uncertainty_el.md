# Experiment 005 – Εκτίμηση Οπτικής Αβεβαιότητας

## Στόχος

Στόχος του πειράματος είναι η μετατροπή της ποιότητας χαρακτηριστικών σε μέτρο οπτικής αβεβαιότητας.

Η βασική υπόθεση είναι ότι χαμηλή ποιότητα χαρακτηριστικών υποδηλώνει αυξημένη αβεβαιότητα στην εκτίμηση κίνησης.

## Μεθοδολογία

Η ποιότητα χαρακτηριστικών κανονικοποιείται στο διάστημα [0,1].

Στη συνέχεια ορίζεται δείκτης οπτικής αβεβαιότητας αντιστρόφως ανάλογος της ποιότητας.

## Αποτελέσματα

Παράχθηκε χρονοσειρά οπτικής αβεβαιότητας για ολόκληρη την ακολουθία MH_01_easy.

## Συμπέρασμα

Οι μεταβολές της ποιότητας χαρακτηριστικών μπορούν να χρησιμοποιηθούν ως δείκτης αξιοπιστίας του οπτικού υποσυστήματος.
